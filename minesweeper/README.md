# Minesweeper

A version of the classic Minesweeper game built using [React](https://reactjs.org/) and [Redux](https://redux.js.org/).

![Demo](https://imgur.com/y6FGSST.gif)

## Getting Started

```
cd minesweeper
npm install
npm run build
```
Then navigate to [http://localhost:3000](http://localhost:3000).
