# Tic-Tac-Toe

A two-person tic-tac-toe game with win tally, built using vanilla Javascript and [Express](http://expressjs.com/).

![Demo](https://imgur.com/1C1nFuI.gif)

## Getting Started

```
npm install
npm run start
```
Then navigate to [http://localhost:3000](http://localhost:3000).
